-------------------------------------------
Petal Particles - Cherry Petals
-------------------------------------------

This is a very beautiful & realistic cherry petals particle system for Unity.

Contains:
- 1 Animated texture.
- 1 demo scene.
- 2 particle prefabs.

The demo scene are located in the folder "\Assets\Petal Particles - Cherry Petals\Demo Scene\".

This project has been tested on unity 2017.1.0/ 2017.4.29/ 2018.4.3/ 2019.1.5/ 2019.2.12.
